# Ícones e assets pendentes

Coloque nesta pasta os seguintes arquivos antes do build de produção
(referenciados em `vite.config.ts` e `index.html`):

- `favicon.ico`
- `logo.png`
- `pwa-192x192.png` (192×192)
- `pwa-512x512.png` (512×512)

Enquanto não existirem, o app roda em dev normalmente (apenas 404 no favicon).
Gerar a partir da identidade visual MedSest (azul marinho #1A3A5C).
